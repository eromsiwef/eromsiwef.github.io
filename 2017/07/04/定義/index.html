定義

sudo的在英语里的意思是switch user and do something

PPA基本上就是一些官方沒收錄或是遠比官方還新的套件來源
但不用開發者自己架apt server而是丟在Launchpad，而且很容易新增/移除repository

用法：
sudo add-apt-repository ppa:PPA_NAME
(或是搭配add-apt-ppa使用)
