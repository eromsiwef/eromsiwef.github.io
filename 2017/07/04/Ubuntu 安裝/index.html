Ubuntu 安裝

安装ubuntu

  • Name and operating system
    Name: type any
    Type: Linux
    Version: Ubuntu(64-bit)

  • Memory size
    8092 MB

  • Hard disk
    Create a virtual hard disk now

  • Hard disk file type
    VDI

  • Storage on physical hard disk
    Dynamically allocated

  • File location and size
    you should create a new empty directory name like “vb”
    in Document directory, and then select 100 GB for disk

========================================================

掛載映像檔和啟動

  • Settings -> Storage -> 選擇光碟片icon
    Choose Virtaul Optical Disck File
    選擇下載的 Ubuntu ISO

  • Start

========================================================

  • 安裝 Ubuntu
    進入到安裝畫面，選擇語系後，點「安裝 Ubuntu」
    可勾選 為圖像及 Wi-FI … -> 繼續
    勾選 清除磁碟並安裝 Ubuntu -> 立刻安裝
    預設時間 -> 繼續
    鍵般選 英語(美式) -> 繼續
    輸入名稱、電腦名稱、使用者名稱和密碼，選擇「自動登入」，點選「繼續」
    密碼：mall89
    安裝中…
    安裝完成，點選「立即重新啟動」
    VirtualBox無法重新啟動，點選「機器」的「關閉」
    選擇「機器關機」，點選「確定」，關閉虛擬機器
    虛擬機器關閉以後，點選「啟動」，就可以重新開啟Ubuntu 16.04

========================================================

##Ubuntu 一些設定

  • 雙向複製貼上設定方法：
    VirtualBox沒有掛套件，單純設定雙向剪貼是沒用的
    需要把 把Guest-Additions裝起來

裝置 -> 插入 Guest Additions CD image
安裝…然後重啟
然後在到VirtualBox上點選
裝置-> 共用剪貼簿-> 雙向
即可複製文字，剪下貼上

  • Ubuntu : vi 裡”上下左右” 變成 “ABCD” !!

安裝 Oracle JDK 8

  • 首先新增一個 PPA

列出目前系統預設的 Java 及所有可用的 Java

可以得知目前 Java 安裝入徑在 /usr/lib/jvm/java-8-oracle/jre/bin/java

設定 JAVA_HOME
